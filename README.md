# Pineapple
A ProjectManagement tool On React

## 技术栈

React+React-router+Redux+Webpack+AntD+React-Bootstrap+Node.js+Sequelize+Express.js+Mysql

## 安装依赖&运行：
### 客户端(react)
- cd ./react
- npm install
- npm start
### 服务端(node)
- cd ./server
- npm install
- npm start

## 客户端发布(react部分)：
- npm run build
